In this week, we focused on the following key areas:

In this week, we focused on the following key areas:

- Understanding the Model-View-Controller (MVC) design pattern
- Creating ASP.NET Core projects using Visual Studio
- Exploring server-client architecture and communication
- Building basic MVC applications with Razor Pages
- Developing and debugging projects using .NET tools

## Projects Included

- **Project 1: ASP.NET Core Empty Project**  
  Practiced creating an empty ASP.NET Core project using Visual Studio:
  - Explored project folder structure and default configurations
  - Implemented basic routing to understand request-response handling

- **Project 2: Basic MVC Project**  
  Developed a simple MVC application with Razor Pages:
  - Implemented a model for handling data
  - Built a view to display dynamic content
  - Created a controller to manage user interactions and responses
  - Tested CRUD operations with dynamic inputs and routing
